This web application is for Currency Conversion.
The only thing that you need to do is to put your intended amount, type the currencies that you want to convert from and into, and in the end, click on the Convert button
Note that the currencies that can be supported by the application are:
1) Euro
2) Dollar (which is U.S. Dollar)
3) Sek (Which is Swedish Krona)
4) Ruble (which is Russian Ruble)

Example: Amount: 100, From: Euro, To: Dollar
